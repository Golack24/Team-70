# Team-70 Presents METRIC

<img width="907" height="247" alt="image" src="https://github.com/user-attachments/assets/f3ffb6b9-042f-4e2d-a231-4e691280a8ec" />

Metric is a full-stack e-commerce web application designed for browsing, purchasing, and managing high-performance gymwear. The system focuses on clean UI design, responsive layouts, and robust backend integration using a relational database.

This project was developed as a team-based coursework project, with responsibilities divided across frontend development, backend logic, and database design.

## Features

###  Product Browsing
- Browse products by category (Men, Women, Accessories)
- Clean grid-based UI for product display
- Consistent design across pages

###  Search Functionality
- Search products by name or description
- Supports scalability as product catalogue grows

###  Basket System
- Add items to cart
- Update quantities
- Persistent cart linked to user via database

###  Checkout System
- Displays cart items and total price dynamically
- Includes discount/coupon input field
- Structured layout (cart summary + checkout section)

###  User Authentication
- User registration and login system
- Session-based authentication
- Role support (customer, admin, staff)

###  Database Integration
- MySQL relational database
- Key tables:
  - Users
  - Products
  - Categories
  - Basket
  - Orders
  - Shipping
- Uses foreign keys to maintain data integrity

###  Order & Shipping
- Orders linked to users
- Shipping table supports tracking and delivery status

###  Coupons System
- Database structure for discount codes
- Supports percentage and fixed discounts

###  Chatbot Integration (Planned / Optional)
- Can be integrated for:
  - Product recommendations
  - Customer support
  - Order tracking assistance

###  Responsive Design
- Mobile-friendly layout
- Adaptive navigation and UI components
